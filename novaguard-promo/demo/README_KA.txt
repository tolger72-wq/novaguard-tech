═══════════════════════════════════════════════════════════
  NOVAGUARD PROMO ENGINE — დემო პაკეტი
═══════════════════════════════════════════════════════════

ამ საქაღალდეში ყველაფერი დემონსტრაციის/პრეზენტაციისთვისაა.

სწრაფი დაწყება (backend-ის გარეშე, მხოლოდ ვიზუალური დემო):
─────────────────────────────────────────────────────────
1. ორმაგი დაწკაპუნება novaguard-admin.html-ზე → იხსნება ადმინ პანელი
2. ორმაგი დაწკაპუნება novaguard-display.html-ზე → იხსნება დიდი ეკრანი
   (გადაიტანეთ ეს ეკრანი მეორე მონიტორზე/ტელევიზორზე, სრულ ეკრანზე - F11)

ორივე ეკრანი მუშაობს backend-თან კავშირის გარეშეც — აჩვენებს
"დემო რეჟიმს", ჩანართები დასათვალიერებელია, ღილაკების ანიმაციები მუშაობს.


სრული სისტემის გაშვება (რეალური მონაცემები + ავტომატური გათამაშება):
─────────────────────────────────────────────────────────
1. საჭიროა Docker Desktop-ის ინსტალაცია (docker.com/products/docker-desktop)
2. გახსენით ტერმინალი, შედით novaguard-promo საქაღალდეში
3. გაუშვით შემდეგი ბრძანებები:

   cp .env.example .env
   ./quickstart.sh

4. რამდენიმე წუთში სისტემა მზად იქნება:
   - API:        http://localhost:8000
   - API დოკუმენტაცია: http://localhost:8000/docs
   - ადმინ პანელი: გახსენით admin.html, დააჭირეთ "Bağlan" ღილაკს
   - ეკრანი:     გახსენით display/display.html, "ოპერატორის"
                 პანელში შეიყვანეთ API მისამართი, დააჭირეთ Connect

5. quickstart.sh ავტომატურად ტვირთავს 10 დემო მოთამაშეს და 3
   მაგალით გათამაშებას — ადმინ პანელში მაშინვე დაინახავთ.


ფაილების გზამკვლევი:
─────────────────────────────────────────────────────────
novaguard-admin.html      → ადმინისტრატორის პანელი (ერთი ფაილი, ორმაგი დაწკაპუნება)
novaguard-display.html    → დიდი ეკრანი / ტელევიზორი (ერთი ფაილი, ორმაგი დაწკაპუნება)
ornek-sertifika.pdf       → გათამაშების სერტიფიკატის რეალური მაგალითი
novaguard-promo/          → სრული პროგრამული კოდი (backend + ტესტები)

  novaguard-promo/README.md         → ტექნიკური დოკუმენტაცია
  novaguard-promo/quickstart.sh     → ერთი ბრძანებით ინსტალაცია
  novaguard-promo/branding.json     → კაზინოს ლოგო/სახელი/ფერის პარამეტრები
  novaguard-promo/scripts/          → კაზინოს ინტეგრაციის მაგალითები
  novaguard-promo/tests/            → 38 ავტომატური ტესტი (ყველა წარმატებული)


კითხვებისთვის:
─────────────────────────────────────────────────────────
README.md ფაილში დეტალურადაა აღწერილი API-ის გამოყენება,
კაზინოსთან ინტეგრაცია და საგადასახადო დეკლარაციის პროცესი.

═══════════════════════════════════════════════════════════
  გადატანა USB / ფლეშ დისკით — გაშვება ინტერნეტის გარეშე
═══════════════════════════════════════════════════════════

თუ გსურთ ეს პროექტი USB დისკზე გადაიტანოთ და გაუშვათ ისეთ
კომპიუტერზე, სადაც ინტერნეტი არ არის (კაზინოს საკუთარი
სერვერის ჩათვლით), საჭიროა ორი ეტაპი:

ეტაპი 1 — მომზადება (მხოლოდ ერთხელ, ინტერნეტიან კომპიუტერზე)
─────────────────────────────────────────────────────────
1. კომპიუტერზე, სადაც დაინსტალირებულია Docker Desktop და არის
   ინტერნეტი, გახსენით novaguard-promo საქაღალდე
2. ტერმინალში გაუშვით:

     python3 prepare_offline_package.py

3. ეს გასტანს რამდენიმე წუთს და შეიქმნება საქაღალდე
   სახელწოდებით "docker-images"
4. დააკოპირეთ მთელი novaguard-promo საქაღალდე (docker-images-ის
   ჩათვლით) USB დისკზე


ეტაპი 2 — გაშვება (სამიზნე კომპიუტერზე, ინტერნეტი არ არის საჭირო)
─────────────────────────────────────────────────────────
1. დააკოპირეთ novaguard-promo საქაღალდე USB დისკიდან კომპიუტერზე
2. ამ კომპიუტერზეც უნდა იყოს დაინსტალირებული Docker Desktop
   (ამ ინსტალაციასაც სჭირდება ინტერნეტი, მაგრამ ინსტალაციის
   შემდეგ NovaGuard თავად მუშაობს ინტერნეტის გარეშე)
3. ტერმინალში, საქაღალდის შიგნით, გაუშვით:

     python3 load_offline_images.py

4. სისტემა ავტომატურად ეშვება — შეგიძლიათ გახსნათ admin.html
   და display.html ფაილები ჩვეულებრივად


შეჯამება — რას სჭირდება ინტერნეტი, რას არ სჭირდება?
─────────────────────────────────────────────────────────
✓ admin.html / display.html / kiosk.html  → არასოდეს სჭირდება ინტერნეტი
✓ Docker Desktop-ის ინსტალაცია              → ინტერნეტი სჭირდება მხოლოდ პირველად
✓ prepare_offline_package.py               → სჭირდება ინტერნეტი (მხოლოდ ერთხელ, მოსამზადებელ კომპიუტერზე)
✓ load_offline_images.py                   → არ სჭირდება ინტერნეტი (სამიზნე კომპიუტერზე)
